Get a chatbot up and running NOW!

  1. Click here to
  2. Set Heroku Secret Variables to gain access to their service
  3. GOOGLE_WEB_CLIENT_ID and GOOGLE_WEB_CLIENT_SECRET with Google OAuth2 Keys instructions are needed for user login and to connect Google Drive to their chatbot.
  4. OPENAI_API_KEY for ChatGPT4.
  5. HUGGINGFACE_API_KEY to use a HuggingFace LLM.


App Developers


You can clone the repo and get started with our demo chatbot

  1. Download Repo

  2. npm install

  3. Create Google OAuth2 Keys
  4. .env.examples -> .env + Keys
  5. npm run start
